# Publishing Checklist

公開前にこのチェックリストを確認します。

## 必須チェック

- [ ] APIキー、トークン、パスワードが含まれていない
- [ ] 秘密鍵、証明書、認証情報が含まれていない
- [ ] 個人情報、顧客情報、契約情報が含まれていない
- [ ] 本番DB接続情報、内部管理画面URLが含まれていない
- [ ] 脆弱性の詳細や攻撃手順が含まれていない
- [ ] 内部向けの運用手順や障害対応詳細が含まれていない
- [ ] 画像やスクリーンショットに秘密情報が映っていない
- [ ] 外部公開してよい抽象度になっている
- [ ] Notionの `Publish OK` がチェックされている

## 公開NGの例

```text
.env の中身
GitHub Personal Access Token
AWS Access Key
DB接続文字列
管理画面URL
顧客名やメールアドレス
未公開ロードマップ
障害対応の詳細ログ
```

## 公開OKの例

```text
プロジェクト概要
セットアップ手順
公開APIの使い方
FAQ
リリースノート
公開してよい利用ガイド
```
